# AI Image Generator using Python

This project is a simple AI Image Generator built using Python. It uses the Pollinations AI image generation API to create images from text prompts.

## 📌 Features

* Generate AI images from text descriptions
* Simple and beginner-friendly Python code
* Automatically saves generated images
* Uses `requests` library for API handling

## 🛠️ Technologies Used

* Python
* Requests Library
* Pollinations AI API

## 📂 Project Structure

```bash
├── imgcreate.py
└── README.md
```

## 🚀 How It Works

1. Enter a text prompt in the Python script.
2. The prompt is sent to the AI image generation API.
3. The generated image is downloaded automatically.
4. The image is saved as `y.png`.

## ▶️ Installation & Usage

### Step 1: Install Python

Download and install Python from:

[Python Official Website](https://www.python.org?utm_source=chatgpt.com)

### Step 2: Install Required Library

```bash

```

### Step 3: Run the Program

```bash

```

## 💡 Example Prompt

```python
prompt = "Girl sitting on a bench in a park, surrounded by blooming flowers and trees, with a serene expression on her face."
```

## 📸 Output

The generated image will be saved as:

